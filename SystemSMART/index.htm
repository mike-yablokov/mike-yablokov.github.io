Система SMART

Программный комплекс состоит из двух программ, это приложение для Windows на основной компьютер и мобильные приложения для Android на смартфоны инспекторов. Сама программа бесплатная, к ней есть описание, инструкции и техническая документация на сайте (на английском): https://smartconservationtools.org/

Зачем нужен SMART на ООПТ?

- для простого и удобного сбора данных на территории;

- для контроля сотрудников;

- для автоматического создания отчётов по заранее определённым формам;

- для аналитики работы службы охраны и повышения её эффективности.

Презентация о системе СМАРТ (pdf-файл презентации).

Я занимаюсь полным комплексом работ по внедрению системы в работу ООПТ, от разработки базы данных до обучения сотрудников.

Я проходил обучение по развёртыванию системы SMART в Норвегии, в Полярном Институте города Тромсё.

Для создание системы необходимо подготовить формы отчётов, которые она должна будет генерировать, это фактически будет техзаданием. По этим формам я создам базы данных, настрою приложения. Затем я устанавливаю и настраиваю программное обеспечение на месте, провожу обучающий семинар для инспекторов и сотрудников, ответственных за сбор данных.

Полный комплекс работ я планировал и проводил на нескольких территориях: для охотуправления республики Якутия (проект WWF по сохранению дикого северного оленя), В нацпарке Берингия, по медвежьим патрулям на Чукотке, в природном парке “Ергаки”, Красноярский Край, в Ремдовском заказнике и Себежском национальном парке (Псковская область).
