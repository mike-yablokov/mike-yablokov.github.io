Политика конфиденциальности SimplePDFReader

Дата последнего обновления: 16 ноября 2025 г.

1. Общие положения

Настоящая Политика конфиденциальности определяет порядок обработки и защиты персональных данных пользователей мобильного приложения SimplePDFReader (далее — "Приложение"), разработанного Михаилом Яблоковым (далее — "Разработчик").

Используя Приложение, вы соглашаетесь с условиями настоящей Политики конфиденциальности.

2. Сбор данных

Приложение SimplePDFReader не собирает и не передает персональные данные пользователей. Приложение работает полностью офлайн и не требует подключения к интернету для основной функциональности.

Приложение не использует аналитику, не отслеживает действия пользователей и не передает данные на внешние серверы.

3. Локальное хранение данных

Приложение хранит следующие данные только на устройстве пользователя в локальном хранилище:

  • Настройки приложения (предпочтения пользователя)
  • История открытых файлов (опционально)
  • Флаги состояния (например, статус заполнения анкеты)

Все данные хранятся исключительно на устройстве пользователя и не передаются третьим лицам.

4. Обработка платежей

Приложение использует RuStore Pay SDK для обработки платежей (донатов). Обработка платежей происходит через платформу RuStore в соответствии с их политикой конфиденциальности.

Разработчик не получает и не хранит данные банковских карт или другую платежную информацию. Вся информация о платежах обрабатывается исключительно RuStore.

5. Анкета обратной связи

Приложение может содержать ссылку на анкету обратной связи (Яндекс.Формы). Заполнение анкеты является добровольным. Данные, предоставленные в анкете, обрабатываются в соответствии с политикой конфиденциальности Яндекс.Форм.

Разработчик получает только те данные, которые пользователь добровольно указывает в анкете. Если пользователь добровольно указывает адрес своей электронной почты для обратной связи, это означает его согласие на передедачу, обработку и хранение его персональных данных.

6. Права пользователя

Пользователь имеет право:

  • Удалить приложение в любое время
  • Очистить данные приложения через настройки устройства
  • Не заполнять анкету обратной связи
  • Не совершать платежи (приложение полностью бесплатное)

7. Изменения в Политике конфиденциальности

Разработчик оставляет за собой право вносить изменения в настоящую Политику конфиденциальности. Актуальная версия всегда доступна на данной странице. Дата последнего обновления указана в начале документа.
